CP TutsPaging

Show All

Tutspaging menggunakan sistem tabs milik Bootstrap, jadi ada dependensi ke bootstrap.

3 elemen utama pada paging ini:

  • Artikel
  • Pagination
  • Link showall
Pada step selanjutnya dijelaskan cara integrasi ke sistem.

Assets

Embed file tutspaging.css di head dan tutspaging.js di footer.

Artikel

Artikel dibungkus di dalam elemen article atau elemen lainnya dan menambahkan class .tutspaging padanya. Artikel tersusun dari judul, konten dan pagination.

Konten artikel harus dikemas dalam container berclass .step-contents .tab-content. Pada konten artikel harus ada pemisahan konten per pagenya, menggunakan class .tab-pane .step.

Pagination

Tambahkan kode pagination ini di bagian bawah setelah elemen .step-contents:

<ul id="step-pagination" class="nav nav-pills">
	<li id="pagination-loading">
		<img style="margin:0;" src="http://i.imgur.com/695GlT0.gif">
	</li>
</ul>

Showall Link

Link ini dapat disimpan dimana saja di halaman:

<a href="#showall" id="showall-toggle">Show All</a>

Bagian yang wajib ada dari link ini adalah atribut href, id dan captionnya. Bila ingin mengganti caption, maka gantilah di dalam file js.
